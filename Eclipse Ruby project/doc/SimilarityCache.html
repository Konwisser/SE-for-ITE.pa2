    class SimilarityCache

    This class is used to cache already calculated similarities between two users as well as the “most similar users” lists for particular users.

    Georg Konwisser (software@konwisser.de)

def get_sim(user1, user2)
        user1.id < user2.id ? @sim[user1][user2] : @sim[user2][user1]
end

def put_sim(user1, user2, sim)
        user1, user2 = user2, user1 if user1.id > user2.id
        @sim[user1][user2] = sim
end
