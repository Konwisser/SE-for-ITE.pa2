    class Movie

    This model class represents a single movie and contains references to all its ratings.

    Author

    Georg Konwisser (software@konwisser.de)

    Attributes

    id[R]

    the numeric id of this movie as provided in the data set

    popularity[RW]

    A numeric value representing the popularity of this movie. A higher value means a greater popularity.

    ratings[R]

    a list of ratings ( Rating objects ) for this movie
